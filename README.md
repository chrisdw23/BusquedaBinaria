BusquedaBinaria
¿Qué hace?
Este programa busca un número dentro de una lista ordenada usando el algoritmo de Búsqueda Binaria

Asi funciona:
- Se toma el número del medio de la lista
- Si es el número buscado → queda todo listo
- Si es mayor → se busca en la mitad izquierda
- Si es menor → se busca en la mitad derecha
- El proceso se repite hasta encontrar el número o hasta que no haya más elementos

Ejemplo
Lista:
3, 7, 10, 14, 18, 22, 27, 30, 35, 40, 45, 50

El número a buscar es el 27, como en Java los indices comienzan desde 0 entonces la posición de ese numero es la 6
